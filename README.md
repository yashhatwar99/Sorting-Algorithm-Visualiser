## ğŸ® Controls

| Key     | Action                              |
|---------|-------------------------------------|
| **B**   | Run Bubble Sort                     |
| **S**   | Run Selection Sort                  |
| **I**   | Run Insertion Sort                  |

| **Close Window** | Quit the program           |
